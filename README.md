## Customization

### Change Profile Image

Replace the placeholder avatar in `index.html` with your actual photo:

```html
<div class="image-frame">
    <img src="your-photo.jpg" alt="Gabby Brown" style="width: 100%; height: 100%; object-fit: cover; border-radius: 26px;">
</div>
```

### Update Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --pink-primary: #ff8fab;
    --pink-medium: #ff6b95;
    --pink-dark: #e84a7f;
    /* ... other colors */
}
```

### Update Content

Edit text directly in `index.html` to update:

- Stats numbers
- About cards
- Content descriptions
- Social links
